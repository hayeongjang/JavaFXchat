# Java Chatting
## 프로젝트 핵심
1. 유튜브 강의를 통해 클린 코드 형식으로 제작하였다. 
2. 전체적인 네트워크 통신 흐름을 알기 위해 진행하였다.
3. 이론 공부를 통해 얻은 지식보다 프로젝트를 진행하며 모르는 것을 그때그때 찾아가며 학습하는 것이 훨씬 효율적이었다!!
## 개발도구
JDK, e(fx)clipse plug-in, eclipse
## 개발과정
